# NN Events Bot 🤖

Telegram-бот для поиска IT и развлекательных мероприятий в Нижнем Новгороде с социальными функциями.

## 📱 О проекте

NN Events Bot помогает находить интересные события в Нижнем Новгороде и планировать посещение вместе с друзьями.

### Что умеет бот:

🎯 **События**
- Просмотр актуальных IT-мероприятий и развлекательных событий
- Фильтрация по категориям
- Детальная информация о каждом событии

📆 **Календарь**
- Просмотр событий на сегодня, завтра, неделю или месяц
- Удобная группировка по дням

👥 **Друзья**
- Добавление друзей по username
- Приглашение друзей по реферальной ссылке
- Просмотр кого из друзей идет на событие

✅ **Планирование**
- Отметка "Я пойду" или "Возможно" на события
- Приглашение друзей на интересные события
- Отмена записи в любой момент

🔔 **Уведомления**
- Узнавайте когда друзья отмечаются на события
- Настраивайте типы уведомлений в профиле
- Быстрая запись на событие прямо из уведомления

👤 **Профиль**
- Статистика: количество друзей и запланированных событий
- Настройки уведомлений
- Управление приватностью

## 🚀 Как начать

1. Найдите бота в Telegram: **@NN_HSE_Events_Bot**
2. Нажмите `/start`
3. Готово! Можете смотреть события и добавлять друзей

## 💡 Основные команды

- `/start` - Начать работу с ботом
- `/events` - Посмотреть все события
- `/calendar` - Открыть календарь
- `/friends` - Управление друзьями
- `/profile` - Ваш профиль и настройки
- `/help` - Справка по командам

## 🎯 Категории событий

- **💻 IT** - Конференции, митапы, хакатоны, воркшопы
- **🎉 Развлечения** - Концерты, фестивали, встречи, выставки

## 🛠️ Технологии

- Python 3.10
- aiogram 3.x (Telegram Bot API)
- PostgreSQL 15
- Redis
- Docker Compose
